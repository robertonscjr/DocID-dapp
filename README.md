# DocID-dapp

## Install tools

Truffle:
```bash
npm install -g truffle
```

Ganache:
```bash
npm install -g ganache-cli
```

https://www.trufflesuite.com/

## Deploy contract

Running ganache:
```bash
ganache-cli
```

Compiling the contract:
```bash
 truffle compile
 ```
 
 Deploying the contract:
```bash
 truffle migrate
 ```
